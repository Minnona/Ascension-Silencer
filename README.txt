ASCENSION SILENCER

A configurable chat filter for Project Ascension.

Filters:
- Guild recruitment advertisements
- Non-English messages, including Balkan and Turkish text
- Donation Point and Bazaar Token trading spam
- Twitch, Kick and livestream advertisements

Type /as to open settings.

WINDOWS INSTALLATION
Place the AscensionSilencer folder in:
C:\AscensionWoW\resources\ascension-live\Interface\AddOns\

LINUX INSTALLATION
Place the AscensionSilencer folder in:
~/AscensionWoW/resources/ascension-live/Interface/AddOns/

Use your own AscensionWoW location if the game is installed elsewhere.
